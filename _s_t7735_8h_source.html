165 //------------ST7735_DrawBitmap------------
166 // Displays a 16-bit color BMP image. A bitmap file that is created
167 // by a PC image processing program has a header and may be padded
168 // with dummy columns so the data have four byte alignment. This
169 // function assumes that all of that has been stripped out, and the
170 // array image[] has one 16-bit halfword for each pixel to be
171 // displayed on the screen (encoded in reverse order, which is
172 // standard for bitmap files). An array can be created in this
173 // format from a 24-bit-per-pixel .bmp file using the associated
174 // converter program.
175 // (x,y) is the screen location of the lower left corner of BMP image
176 // Requires (11 + 2*w*h) bytes of transmission (assuming image fully on screen)
177 // Input: x horizontal position of the bottom left corner of the image, columns from the left edge
178 // y vertical position of the bottom left corner of the image, rows from the top edge
179 // image pointer to a 16-bit color BMP image
180 // w number of pixels wide
181 // h number of pixels tall
182 // Output: none
183 // Must be less than or equal to 128 pixels wide by 160 pixels high

186 //------------ST7735_DrawCharS------------
187 // Simple character draw function. This is the same function from
188 // Adafruit_GFX.c but adapted for this processor. However, each call
189 // to ST7735_DrawPixel() calls setAddrWindow(), which needs to send
190 // many extra data and commands. If the background color is the same
191 // as the text color, no background will be printed, and text can be
192 // drawn right over existing images without covering them with a box.
193 // Requires (11 + 2*size*size)*6*8 (image fully on screen; textcolor != bgColor)
194 // Input: x horizontal position of the top left corner of the character, columns from the left edge
195 // y vertical position of the top left corner of the character, rows from the top edge
196 // c character to be printed
197 // textColor 16-bit color of the character
198 // bgColor 16-bit color of the background
199 // size number of pixels per character pixel (e.g. size==2 prints each pixel of font as 2x2 square)
200 // Output: none
201 void ST7735_DrawCharS(int16_t x, int16_t y, uint8_t c, int16_t textColor, int16_t bgColor, uint8_t size);
