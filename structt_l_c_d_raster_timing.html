uint8_t tLCDRasterTiming::ui8VSyncWidth

In active mode, a value from 1 to 64 that specifies the number of line clock periods to set the lcd_fp pin active at the end of each frame after the vertical front porch period elapses. The number of The frame clock is used as the VSYNC signal in active mode.

In passive mode, a value from 1 to 64 that specifies the number of extra line clock periods to insert after the vertical front porch period has elapsed. Note that the width of lcd_fp is not affected by this value in passive mode.
